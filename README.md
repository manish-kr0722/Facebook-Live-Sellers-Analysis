# Facebook Live Sellers Analysis 📊

An end-to-end data analysis project designed to evaluate and optimize engagement patterns for Facebook Live sellers. This repository tracks the full data pipeline from raw metrics to actionable business insights.

## 🚀 Project Overview
* *SQL:* Data extraction, cleanup, and advanced metric aggregation.
* *Python:* Exploratory Data Analysis (EDA) and customer segmentation using K-Means clustering.
* *Power BI:* Interactive, multi-page dashboard tracking engagement trends, reaction types, and KPI performance.
